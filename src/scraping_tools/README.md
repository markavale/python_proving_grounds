# Web Scraping Tools

This module contains multiple sub-modules each dedicated to different scraping technologies: Scrapy, BeautifulSoup, Selenium, and Playwright.

## Sub-modules
- `scrapy_projects/` - Projects using Scrapy.
- `beautifulsoup_examples/` - Simple scripts utilizing BeautifulSoup.
- `selenium_scripts/` - Scripts that demonstrate web automation using Selenium.
- `playwright_scripts/` - Examples using Playwright for both scraping and automation.

## Setup and Usage
Follow the README.md in each subdirectory for detailed setup and usage instructions.
