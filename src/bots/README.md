# Python Bots

This directory hosts various bots built for platforms like Twitter, Reddit, Discord, and YouTube.

## Bots
- `twitter/` - A Twitter bot that automates tweeting and interactions.
- `reddit/` - A Reddit bot for moderation and post management.
- `discord/` - A Discord server bot with multiple functionalities.
- `youtube/` - A YouTube bot to automate common channel management tasks.

## Installation
Select the bot directory you're interested in and follow the setup instructions in its specific README.md.
